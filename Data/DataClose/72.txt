Misleading organization number in left navbar in dashboard of system admin
kind/bug team/plugin sprint/next-sprint sprint/next-sprint 
Description Number of organization in left navbar in dashboard of system admin doesn't take into account sub-organizations. Reproduction Steps Open dashboard as system admin. Create organization test. Create sub-organization child. Expected result organization number in left navbar menu equals to number of organization in organizations list = 2. Actual misleading result organization number in left navbar menu = 1 whereas number of organization in organizations list was 2. Solution use the same method to calculate the number of organizations as we use for regular user: getOrganizationsNumber(). OS and version: Eclipse Che Multi User 5.21.0-SNAPSHOT
