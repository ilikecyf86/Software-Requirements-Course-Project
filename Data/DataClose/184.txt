Folders are not visible in the Project Explorer if in the Editor opened a file from root of a project folder
kind/bug severity/P2 severity/P1 severity/P2 team/plugin status/in-progress 
Reproduction Steps: Create a workspace from the Java stack with the web-java-spring project. Open in the Editor README.md or pom.xml file. Close the folder Refresh the web page Observed behavior: After refreshing in the Project Explorer visible only files which opened in the Editor. This issue is similar to #4733 and #5986, except that the web-java-spring folder should be closed before updating. This issue is also reproduced if open a file from the Navigate To File feature.
