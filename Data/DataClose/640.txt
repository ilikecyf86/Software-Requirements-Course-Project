List of libraries does not redraw automaticaly after change pom.xml
kind/bug severity/P2 team/plugin 
After add or remove dependency in the pom.xml, list of libraries will be changed only after click on "Refresh Project Explore" button or close and expand libraries folder or refresh browser . Reproduction Steps: Create java stack with default spring project Open root folder Open folder with libraries Open pom.xml Change dependency (for example remove exists or add new dependency) Pay attention that libraries list does not changed automaticaly OS and version: CHE 5.16.0 ; Chrome 59.0 ; Fedora 25
