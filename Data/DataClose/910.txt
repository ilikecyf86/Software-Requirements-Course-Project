The File Structure widget does not close after pressing Esc button
kind/bug severity/P2 team/ide sprint/next-sprint sprint/next-sprint status/in-progress status/code-review status/in-progress status/code-review 
Reproduction Steps: Create and start a workspace with the web-java-spring project . Open the GreetingController.java file in the Editor. Press Ctr+F12. The File Structure widget appeared. Press Ctr+F12 again and then press Esc button to close the widget. Expected behavior: The File Structure widget closed. Observed behavior: The File Structure widget didn't close. This bug happens only once when open a java file after starting a workspace or refreshing a page. This bug was reproduced by FileStructureBaseOperationTest. Che version: 5.12.0-SNAPSHOT OS and version: Ubuntu 16.04 LTS
