Open of new terminal doesn't show processes tab
kind/bug sprint/current-sprint team/ide 
When try to open a new terminal focus doesn't switches to it, if Processes tab is hidden. Reproduction Steps: Open a workspace Make Processes tab inactive (hide it or open another one: Usage or Events, etc.) Try to open terminal using Menu -> Run -> Terminal or using shortcut Alt+F12 Expected behavior: New terminal will be showed to user Observed behavior: Terminal is opened, but not shown (to see it, is needed to manually switch toPrecesses tab). It is uncomfortable and might confuse users. Che version: 5.0.0-M8-SNAPSHOT OS and version: Ubuntu 16.04 Docker version: Version: 1.12.3 API version: 1.24 Che install: tomcat server (zip) Additional information: Problem can be reliably reproduced, doesn't happen randomly: Yes
