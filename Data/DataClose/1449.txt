The MC editor doesn't start when a java file open in the Editor
kind/bug status/open-for-dev team/enterprise severity/P1 
Reproduction Steps: Create a project with java files(like web-java-spring, web-java-petclinic) Open in the Editor any java file Start the Midnight Commander in the Terminal Select any file and press F4 Expected behavior: Selected in the MC file is opened in the MC editor Observed behavior: Nothing happened. While the java file is open in the Editor F4 button doesn't work. Close the java file or open in the Editor any not java file then F4 will be works. Che version: 5.4.0-SNAPSHOT OS and version: Ubuntu 16.04 LTS Docker version: 1.12.3 Che install: Docker container Additional information: Problem can be reliably reproduced, doesn't happen randomly: [Yes]
