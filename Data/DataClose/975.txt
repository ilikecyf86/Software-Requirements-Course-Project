Add process exit code to the core/process process_died event
kind/task 
According to agent installer specification (5.iii.b & 5.iii.c) 5.iii.b - If installation script exit code 0 agent is treated as booted. 5.iii.c - If installation script exit code differs from 0 agent is treated as failed core/process must provide process exit code in process_died event, so it can be used by agent installer to determine conditions specified above.
